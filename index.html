Text in ADA, Y2018-Text-project 1.4.0

To make text handling workable in a ADA programming language and to introduce Regular Expression based on Unicode. The solution extends native ADA with Wide_Wide_String literals with backslash notation for BS, FF, LF, CR and HT control characters and hexadecimal values. &-operator is replaced with +-operator to make it possible to overload concatenations of string expressions (&-operator works as specified in ADA standard).
