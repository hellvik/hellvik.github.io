Package hierarchy

Y2018 and Y2018.Text
package Y2018 is
    Version:constant String:="2";
end Y2018;

package Y2018.Text is
    Version:constant String:=Y2018.Version & ".5.0";
end Y2018.Text;
The code is in following subpackages:
  • Y2018.text.core - Codepoint and string handling
  • Y2018.text.jets - Regular expression
  • Y2018.text.testq - Test and debug support
  • Y2018.text.util - Support of tools
This package hierarchy document the ADS-files of the packages in the project, and usually not all of the ADS-file or the public part of the package.
