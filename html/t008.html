RightC

Push the result to right. Because we are working with 21-bit characters and all indexing is based on theese 21-bit characters.

function rightC(value:CFix;length:Integer;pad:CFix:=CFix_Space) return CFix;
function rightC(value:String;length:Integer;pad:String:=" ") return String;
function rightC(value:CVar;length:Integer;pad:CFix:=CFix_Space) return CVar;
