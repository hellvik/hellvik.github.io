Read

function read(base:in out Y2018.Text.Wide.InFileUTF8.self;result:out Unbounded_String;replace:String;action:Boolean) return RS_TY
read one legal unicode character from the file and return this character in UTF-8 mode, 1..4 bytes.
  • base
  • result:out Unbounded_String - output is directed to this variable
  • replace:String - character value to use if a byte is not a possible UTF-8 byte. Defaulted to unicode replacement character '�'
  • action:Boolean - if true raise exception. Default FALSE, do nothing
  • return RS_TY - return a result value (RS_TY defined in package Y2018.Text.Wide)
    type RS_TY is (
      RS_NONE, -- OK
      RS_MISS, -- Last UTF-8 character in a file is missing bytes
      RS_ILLEGAL, -- Value in a byte cannot be an UTF-8 character or a part of an UTF-8 character
      RS_EOF -- End of File reached
    );
