Package dependencies

Basic dependencies of packages in Y2018.Text

  1. Core : depends only on Ada standard packages
  2. (Testq,) Jets, Regex : depends on Core, and must be constructed in this order
  3. Comp : precompiler Y2018Text, depends on Jets and Regex
  4. Wide, (Testr) : must be precompiled and depends on Jets and Regex
  5. M and anything else  : depends on Wide and must be precompiled
