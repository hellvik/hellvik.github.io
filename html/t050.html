The "TEST" directory is a place for testing examples and tools of Y2018-Text project

Creating the "TEST" enviroment

To create the content "TEST" directory we have to create following links

  1. bin -> {make}/bin
  2. ExFi.prop -> {make}/txt/ExFi.prop
  3. ExKhi.prop -> {make}/txt/ExKhi.prop
  4. lib -> {make}/lib
  5. UCDlink -> UniCodeData15 (the directory "UniCodeData15" where UCD_15.zip is unzipped in)

Where {make} is the directory where we exploded the tar-file of the project Y2018-Text.
UCD_15.zip is the zip-file from Unicode.org, version 15.
