Subpackages:
  • Y2018.Text.Jets.Compiler - the regular expression compiler implementation
  • Y2018.Text.Jets.Jetstool - tools
  • Y2018.Text.Jets.Matchpack - match implementation
  • Y2018.Text.Jets.Patternpack - pattern implementation
  • Y2018.Text.Jets.Rangevectorpack - codepoint class implementation
Allmost all work is done through the subpackages
  1. Definitions of I_A type and functions and constants based on this type
    • I_A_NO_ELEMENT, I_A type refers to a part of a codepoint or character string, this constant is used for null length reference
    • an array of a array type I_A_ARRAY
    • subIA, 3 functions
    • to_String - publish the indexes as a 'nice' string
  2. Reserved characters in a regular expression and backslash, ( |, (, ), [, ], {, }, ^, $, *, +, ?, ., - , \ ) and grouping characters when followed by backslash
  3. REQ_TY type, used by the type RE_Elem definition as a qualifier of different type of nodes in the syntax tree.
