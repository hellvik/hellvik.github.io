Conversions

  • To21 – convert a String to CFix

  • To8 – convert a CFix string to a Character String

Length

  • Length21 – get length of a charater string (String) but as CodePoint units

  • Length8 – get length of a code point string (CFix) but as Character units

Check validity

  • Is_Valid – is a valid CFix string

  • Is_Valid – is a valid UTF-8 string
