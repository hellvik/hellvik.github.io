function "+"(left:STRING;right:STRING) return STRING
function "+"(left:STRING;right:CHARACTER) return STRING
function "+"(left:CHARACTER;right:STRING) return STRING
function "+"(left:CHARACTER;right:CHARACTER) return STRING
Concatenate string or character to another string or charater. "+" operator is used instead for "&" operator.

CHARACTER can be Character or Wide_Wide_Character.
STRING can be any of String, Wide_Wide_String, Unbounded_String or Unbounded_Wide_Wide_String.
